Space Station 13 - TeguStation - Modules - Types — Var Details - Proc Details

/datum/changeling

Stores changeling powers, chemicals, absorbed DNA, and hivemind ID.

Vars

absorbed_countThe total number of people we've absorbed.
absorbed_dnaA list of all "absorbed DNA" datums we have - these includeactual DNA datums, name, species name, and languages.
absorbed_languagesA list of the languages we've absorbed from others. We can speak and understand these languages fluently.
changelingIDThis is a string used when talking over changeling hivemind, instead of the person's current identity.
chem_chargesChemicals are used to fuel changeling powers. This var tracks how many we currently have.
chem_recharge_rateThis many chemicals will be added every life tick to what we currently have.
chem_storageThe max chemicals we can store at a time.
genetic_damageNot actually clone loss, this is used to represent a cooldown period on using abilities.
genetic_pointsGenetic points are our resources for buying new abilities and upgrades.
is_absorbingIf TRUE, we're currently absorbing another person.
mimicingIf we're mimicking someone else's voice, this value tracks that name and uses it when we speak.
owner_mindThe mind that owns this changeling datum.
purchased_powersA list of the power datums we've purchased.
sting_rangeWhen using sting powers, they have a range of this many tiles.

Procs

EvolutionMenuThe code for the menu that the changeling uses to unlock new abilities. Totally stolen from the player panel.
absorb_DNAAdds a new absorb_dna datum into our stored DNA. Chomp.
add_powerAdds the provided power to this changeling datum's purchased powers.
do_transformTransforms the current mob of this mind to match the DNA of the provided absorbed_dna datum.
get_DNAFetches the absorbed_dna datum from a provided name.
regenerateProvides the per-tick regeneration for chemicals and genetic damage.
update_languagesUpdates our parent mob's knowledge of languages based on what we've absorbed.
