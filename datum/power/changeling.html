/datum/power/changeling

The changeling subtype of /datum/power. Contains a lot of custom logic for chemicals, activation checks, and so on.

Vars

allow_during_lesser_formWhether or not we can use this ability while in lesser form. FALSE by default
allow_incapacitatedIf TRUE, we can use this ability while incapacitated. FALSE by default
genome_costThe cost required for the changeling to unlock this power.
max_genetic_damageIf the the genetic_damage of the owner changeling is higher than this amount, we are unable to use the power
max_statIf our current mob's STAT is above this, then we cannot use it
no_autobuyIf TRUE, this power will not automatically purchase if it costs zero genomes.
required_chemsThe amount of chem_charges we need to activate this power
required_dnaIf we haven't absorbed this many genomes, this power will be unusable

Procs
